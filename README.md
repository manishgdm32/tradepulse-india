# TradePulse India - Stock Trading Platform

Professional Indian stock market trading platform with real-time signals, broker integration (Kite, Upstox), and scalping tools.

## Features

- Real-time NSE/BSE market data
- Trading signals (Scalping, Intraday, Swing)
- Broker integration (Zerodha Kite, Upstox)
- Order management
- Price alerts & notifications

## Tech Stack

- React 18 + TypeScript
- Vite
- CSS Modules

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
